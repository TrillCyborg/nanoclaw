# Weather

Get current weather conditions and forecasts for locations worldwide.

## Features

- Current weather conditions (temperature, humidity, conditions)
- Weather forecasts
- Location-based weather lookup
- Supports city names, zip codes, and coordinates

## Usage

Simply ask about the weather:
- "What's the weather in New York?"
- "Weather forecast for London"
- "How's the weather today?"
- "Temperature in Tokyo"

## Data Source

Uses wttr.in weather service for global weather data.
